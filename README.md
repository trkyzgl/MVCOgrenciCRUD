Herkese Merhaba

Öğrencilerin ve Yazarların .Net Core MVC ile DB'ye eklenmesi, DB den çekilmesi,  güncellenmesi, Silinmesi ve öğrencile üzerinde arama yapılması

Uygulamada C#,JavaScript,html,Css dilleri kullanılmıştır.
